Screws and bolts

The following numbers represent the number of screws, washers and nuts we are using to buildthe GummiArm. Note that you can add some screws but it could result in too much constraint on the plastic

M2x8 mm screws : 188

M2 washers : 162

M2 nuts : 168

M2.5x8 mm screws : 122

M2.5x10mm : 4

M2.5x12mm : 52

M2.5 washers : 146

M2.5 nuts : 108

M3x10 mm with bearing bushing : 3

M3x20mm : 37

M3x50mm : 4

M3 washers : 76

M3 nuts : 41

M6x12mm screws : 8

M6 washers : 8
